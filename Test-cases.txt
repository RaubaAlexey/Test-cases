# Test-cases
https://www.russiantrains.com/

Test-cases:

1. Тест на пустое поле.
	1.1 Перейти на сайт russiantrains.com
	1.2 Выбрать пункт отправления
	1.3 Пункт прибытия оставить пустым
	1.4 Выбрать дату отправления
	1.5 Выбрать количество мест
	1.6 Нажать кнопку "Continue"
Результат: Предупреждение о невыбранном месте прибытия.

2. Тест на поездку в прошлое.
	2.1 Перейти на сайт russiantrains.com
	2.2 Изменить дату на компьютере и обновить страницу сайта
	2.3 Выбрать пункт отправления
	2.4 Выбрать пункт прибытия
	2.5 Выбрать дату из "прошлого"
	2.6 Вернуть дату на компьютере
	2.7 Выбрать количество мест
Результат: "No trains on this route or date."
Ожидалось: Ошибка в дате.

3. Тест на одинаковы пункты прибытия и отправления.
	3.1 Перейти на сайт russiantrains.com
	3.2 Выбрать одинаковые пункт отправления и пункт назначения
	3.3 Выбрать дату
	3.4 Выбрать количество мест
	3.5 Нажать кнопку "Continue"
Результат: Перудепреждение о выборе одинаковых пунктов прибытия и отправления.
	
4. Тест на отрицательное количество пассажиров.
	4.1 Перейти на сайт russiantrains.com
	4.2 Выбрать пункт отправления
	4.3 Выбрать пункт прибытия
	4.4 Выбрать дату отправления
	4.5 В поле "Пассажиры" ввести отрицательное число "-5"
	4.6 Нажать кнопку "Continue"
	4.7 Выбрать пункт "Select seats"
	4.8 Выбрать "seat class"
	4.9 Нажать кнопку "Continue"
Ожидалось: Предупреждение о неверном количестве пассажиров.
Результат: Подсчет отрицательной стоимости билетов и появление критической ошибки 
"The website encountered an unexpected error. Please try again later.".

5. Тест на огромное количество пассажиров.
	5.1 Перейти на сайт russiantrains.com
	5.2 Выбрать пункт отправления
	5.3 Выбрать пункт прибытия
	5.4 Выбрать дату отправления
	5.5 Выбрать количество пассажиров равное огромному числу
	5.6 Нажать кнопку "Continue"
Результат: "No trains on this route or date."
Ожидалось: Предупреждение о большом количестве пассажиров
	
6. Тест на пустое поле возраста детей.
	6.1 Перейти на сайт russiantrains.com
	6.2 Выбрать пункт отправления
	6.3 Выбрать пункт прибытия
	6.4 Выбрать дату отправления
	6.5 Выбрать количество пассажирова, оставить поле возраста детей пустым
	6.6 Нажать кнопку "Continue"
Результат: Предупреждение "Choose children age".
	
7. Тест на символьный возраст.
	7.1 Перейти на сайт russiantrains.com
